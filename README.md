## 📌 Project Overview
**Gradient Generator App** is an interactive frontend tool built using **HTML, CSS, and JavaScript**.  
It allows users to generate and preview custom CSS gradients in real time, helping designers and developers quickly find the perfect background style for their web projects.

This project demonstrates dynamic DOM manipulation, real-time input handling, and live CSS style updates using JavaScript.
